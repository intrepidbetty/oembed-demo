oEmbed API Integration

This page demonstrates how Trovee integrates with Meta's oEmbed API to enrich Instagram content saved by users.

How Trovee Uses oEmbed
  1. A user saves an Instagram post or reel to Trovee via the iOS share extension.
  2. Trovee's backend server calls Meta's oEmbed API to retrieve the post's embed HTML and thumbnail for display within the app.
  3. The embedded content is displayed on the user's personal detail page alongside the original creator's attribution.
  4. The oEmbed API is called server-side to render a front-end view of the saved content. No metadata is redistributed or displayed publicly.
